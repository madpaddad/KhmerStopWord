## KHMER STOP WORD ##

Final Project of Information Web Retrieval

## Goal ##
Design a stop-word removal system specifically in Khmer language. 
It includes idenfitying frequent functional words, analyzing the linguistic roles, and evaluating their impact on IR tasks.

## Steps ##
- Precision
- Recall
- TF-IDF weighting
- Search Ranking

## System Architecture ##
- Input: Khmer text documents
- Text Normalization
- Word Segmentation
- POS Tagging
- Name Entity Recognition
- Domain Classification
- Contextual Analysis Stop-word Scoring
- Decision Engine
- Output: Filtered Khmer text documents
